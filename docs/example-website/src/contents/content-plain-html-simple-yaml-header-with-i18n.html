Simple text

Plain HTML content with YAML header that defines the target path.

We cannot use title and shortTitle in plain HTML contents, but it still makes sense to set them such that we can use the get-title handlebars helper from handlebars contents.

i18n

Translation (i18n) tests - everything is translated to German as we have '<html lang="de">'. Usually the language is defined by the YAML section (path with more than one language), but as we don't specify a language in the YAML section, the language is autodetected. Note that if no language can be autodetected, the fallback language (this.settings.i18n.fallbackLanguage) will be used.

  • Fully working translation, brace syntax: {i18n Hello}
  • Translation falling back to fallback language, XML syntax: Hi
  • No translation available, brace syntax: {i18n Bye}

We can also translate a string that is split on multiple lines:

line 1 line 2

{i18n line 1 line 2 }

Typography

Many things are typographically constrained. For example the last word of a sentence should not be alone on a single line.

Also, numbers and words should not be separated: 42 years should stay on the same line.

Finally, a single letter cannot be left at the end of a line. So we force a single letter to be on next line.

It is also possible to not apply typographic rules: "a b c d 42 years" vs. "a b c d 42 years"

It can also span over multiple lines: 42 years this is a test
